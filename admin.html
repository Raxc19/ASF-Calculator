🌐 Setup Auto Price Scraping

Configure automatic price updates from supplier websites

📋 How to Find CSS Selectors:

  1. Right-click on the price on the website
  2. Select "Inspect Element"
  3. Right-click the highlighted HTML element
  4. Choose "Copy" → "Copy selector"
  5. Paste the selector in the field above

Examples:
• Home Depot: .price-format__main-price
• Lowe's: [data-testid="price-display"]
• Menards: .price-current
